basics

  • To start the program, click on the "FractalExplorer.exe" file.
  • Then to make a fractal, press enter or the "create" button.
  • If you like a fractal you can save it as a file with the "save" button.

general fractal guidelines

  • Increase darkness cautiously, large values for darkness make it run much slower.
  • If you want to make it darker it is often better to adjust the expansion.
  • The "reset" button only changes the dimensions from -2 to 2 for x and y, press it if you are lost.
  • The "c value" is like the seed that the image is grown around. It can also be negative.
  • I would not recommend manually adjusting the camera position, using the buttons is much easier
  • If you use colour presets 7 or 8, darkness won't behave normally

advice for making custom functions for x and y

  • See below the fields for valid things to use in the equations
  • For multiplication you must use "*", so 2x would not work and would need to be 2*x.
  • When using exponents use brackets to avoid errors.
  • You can use both "y" and "x" freely in either field.
  • The smaller the values that a function creates, the slower the program will run.

Hotkeys

  • make fractal = enter
  • zoom in = numpad +
  • zoom out = numpad -
  • move camera left = numpad 4(press num lock if it isnt working)
  • move camera down = numpad 2
  • move camera right = numpad 6
  • move camera up = numpad 8
